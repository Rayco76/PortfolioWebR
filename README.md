# Portfolio Web

Portfolio estático en HTML con una página principal (`index.html`) y una página de detalle de proyecto (`project.html`), preparado para publicarse directamente en GitHub Pages sin proceso de build.

## Verlo en local

Abre `index.html` directamente en el navegador.

## Publicarlo en GitHub Pages

1. Sube el contenido del repositorio a la rama `main`.
2. En GitHub, ve a `Settings` → `Pages`.
3. En `Build and deployment`, elige `Deploy from branch`.
4. Selecciona la rama `main` y la carpeta `/root`.
5. Guarda los cambios y espera a que GitHub publique el sitio.
