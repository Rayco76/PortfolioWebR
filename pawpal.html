02Core Experience

Care → Play → Earn → Customize → Return

Care

Routine interaction, needs, affection and daily touchpoints around the pet.

Play

Arcade mini-games add action, variety and short-session energy to the loop.

Earn

Currency, rewards and unlocks reinforce return motivation without overloading the flow.

Customize

Wardrobe and visual personalization turn rewards into expression and collection goals.

Return

Retention systems connect care, rewards and progression so the game invites one more session.

04Character, Animation & Wardrobe

El personaje es el centro del vínculo con el jugador

PawPal is built around an expressive 3D dog character with animation-driven interactions, gameplay states and a wardrobe system prepared for cosmetic expansion.

3D character integration

Setup pensado para gameplay, cámara, claridad de lectura y rendimiento mobile.

Animation states

Estados dedicados para idle, reacción, juego, retorno y momentos de recompensa.

Expressions / feedback

El personaje responde con timing visual y emocional a la acción del jugador.

Wardrobe anchors

Puntos de anclaje preparados para equipables y variantes visuales futuras.

Cosmetic categories

Separación por tipos de item para simplificar contenido, UI y expansión de catálogo.

Future expansion ready

Estructura pensada para crecer con nuevos cosméticos, estados y eventos.
